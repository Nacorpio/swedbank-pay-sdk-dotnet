# About
`PayEx.Client` is a `netstandard2` library to talk to PayEx direct APIs.

# Supported APIs:
- Vipps (create payment, Vipps authorize, capture, cancel, reversal)
- CreditCard link (create payment, capture, cancel, reversal)
